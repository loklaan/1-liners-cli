## Quick Start

```bash
$ npx 1-liners-cli
```

## Keep It Around

```bash
$ npm install -g 1-liners-cli      # Install globally
$ 1-liners                         # Show a list of 1-liners
$ 1-liners isFunction              # Copy a function immediately
$ 1-liners isFunction --print      # Print to stdout instead of copy to clipboard
```
